# Codex Skills

This repository packages local Codex skills for GitHub distribution.

## Skills

- `skills/project-initialization`
- `skills/risk-strategy-metrics`

## Install

Use the Codex skill installer with this repository and the target skill path:

```bash
scripts/install-skill-from-github.py --repo <owner>/<repo> --path skills/project-initialization
scripts/install-skill-from-github.py --repo <owner>/<repo> --path skills/risk-strategy-metrics
```

## Layout

```text
skills/
  project-initialization/
  risk-strategy-metrics/
```
